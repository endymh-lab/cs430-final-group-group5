# Cuisine Tutorial — An Interactive Cooking Website

## Project Description
Cuisine Tutorial is an interactive multimedia web application built as a final group project for CS430 Internet Multimedia Programming. The website allows users to explore and learn how to cook dishes from three different world cuisines Italian, Korean, and North African. Each cuisine features three signature dishes with descriptions, ingredients, and step-by-step cooking instructions.
